BRGAS - Blind Road Assistance and Guidance System

April 2017

Microsoft Code.Fun.Do

This project was done as an entry to Microsoft’s Code.Fun.Do Hackathon, held at our institution. We placed second in the city round, and were invited for Showcase at Microsoft’s India Development Center, at Hyderabad. This system aimed at helping blind people recognise the road signs, and also be able to feel the emotions of the person standing opposite to him. My work involved using clmtrackr library, to find the contours of the facial muscles, and map it to the respective emotions (Happy, Sad, Angry, Surprised). Logistic Regression was used to train the model in order to successfully identify the emotions.

EnTagger

CIP Lab Project

Dalal Bull

December 2016 - January 2017

Stock Market Simulator

This Web Application was built with an aim to create awareness and make it easier for people to understand the working of the Stock Market. We built the application in Python with Redis as database, and hosted it on a Python Server. We used Fuzzy Logic, and Markov Chains for determining the generation of the news, and determining the price of stocks after each transaction. The server was tested on an initial corpus of 250-300 people. The charts was generated using Pygal Library, and rendered as a Vector in the front end. The stock prices were modeled based on the news generated. The prices fluctuated based on Random Motion.

YouTree

August 2017 - November 2017

A Visualization Paradigm of Statistically and Textually Similar Videos

Youtube offers multiple features like number of views, likes, comments and comments text, description text etc for all members through YouTube Data API. Using these direct features, we formulate derived or indirect features which capture much more subtle details about the videos which these direct features couldn’t. We then perform Exploratory Data Analysis on these derived features to analyse the statistical trend of these videos. Next, we find statistically and textually similar videos for a given query video (video under evaluation) and combine them to get a final list of similar videos. We find influential words between these set of similar videos with the query video using Influential Term Metric. Finally, we construct YouTree, a visualisation paradigm that consists of the statistical and textual similar videos for a given root query video.

IBLP: A Framework for Robotic Movement

August 2017 - Going on

Instruction-Based Logic Processing for Mobile Robots

We have worked on a framework for Instruction-Based Logical Processing which uses instructions to bridge the gap between high-level instructions and low-level control. In our approach, simple English is used as an input language for obtaining robotic instructions which are then classified into one of the different types of imperatives based on MIRA (Mimamsa Inspired Representation of Actions). Depending on the kind of imperative, appropriate robot controllers are evoked that controls various sensors and the actuators of the robot. The map for the mobile robot is obtained in the form of regions, and a graph of regions is generated. Motion planning is ap- plied on locomotive instructions to determine the path for the robot and to drive the robot on this route and also to detect and avoid any obstacle in the pursuit. The proposed approach is tested by implementing the software module on robot simulator and on mobile robot constructed using Raspberry Pi.
